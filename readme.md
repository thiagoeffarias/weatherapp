## Weather App - Compose

#### This is and app developed for practice purposes.

##### It makes use of:

* Kotlin
* Jetpack Compose
* Dagger Hilt for dependency Injection
* Coroutines
* Retrofit
* Location Service
* Clean Code
* MVI
* Repository pattern
